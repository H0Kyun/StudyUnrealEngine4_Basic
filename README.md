블루프린트<br>
블루프린트는 C++클래스를 상속받아 제작 가능하지만 그 역은 안된다.
블루프린트가 직관적인 것을 이용하기 위해 C++로 클래스를 만들고 그 클래스를 상속받는 블루프린트를 만든다.<br>
<br>
UI<br>
